# 🚀 Lộ Trình Học Tập Data Analyst (Phân Tích Dữ Liệu)

Chào mừng bạn đến với hành trình trở thành Data Analyst! File này được dùng để tổng hợp lộ trình, tài liệu và theo dõi tiến độ học tập của bạn. 

Dưới đây là các kỹ năng cốt lõi đã được chia thành các thư mục tương ứng. Bạn có thể đánh dấu (x) vào các mục đã hoàn thành nhé.

---

## 📊 1. Excel (Thư mục: `01_Excel`)
Excel là kỹ năng nền tảng nhất của mọi Data Analyst, dùng để xử lý và phân tích dữ liệu nhanh.
- [ ] Các hàm cơ bản & nâng cao (VLOOKUP, HLOOKUP, INDEX, MATCH, IF, logic...).
- [ ] Xử lý và làm sạch dữ liệu (Data Cleansing, Xóa trùng lặp, Text to Columns).
- [ ] **Pivot Table** & Pivot Chart (Cực kỳ quan trọng để tóm tắt dữ liệu).
- [ ] Power Query (Tự động hóa quá trình lấy và làm sạch dữ liệu).

## 🗄️ 2. SQL (Thư mục: `02_SQL`)
SQL dùng để giao tiếp với cơ sở dữ liệu. Hầu hết dữ liệu của doanh nghiệp đều nằm trong các hệ quản trị CSDL.
- [ ] Các câu lệnh truy vấn cơ bản: `SELECT`, `FROM`, `WHERE`, `ORDER BY`, `LIMIT`.
- [ ] Lọc và gom nhóm: `GROUP BY`, `HAVING`, các hàm Aggregate (`SUM`, `COUNT`, `AVG`, `MAX`, `MIN`).
- [ ] Nối bảng (**JOINs**): `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL JOIN`.
- [ ] Phân tích nâng cao với **Window Functions** (`ROW_NUMBER()`, `RANK()`, `OVER()`, `PARTITION BY`).
- [ ] Subqueries & CTEs (`WITH`).

## 📈 3. Data Visualization / BI Tools (Thư mục: `03_Data_Visualization`)
Trực quan hóa dữ liệu giúp kể câu chuyện từ những con số (Data Storytelling). Các công cụ phổ biến: **Power BI** hoặc **Tableau**.
- [ ] Kết nối dữ liệu từ nhiều nguồn khác nhau.
- [ ] Data Modeling (Xây dựng mô hình dữ liệu, quan hệ 1-n, n-n).
- [ ] Viết các hàm DAX cơ bản (nếu dùng Power BI).
- [ ] Tạo các biểu đồ phù hợp (Bar chart, Line chart, Scatter plot, Histogram...).
- [ ] Xây dựng Interactive Dashboard chuyên nghiệp.

## 🐍 4. Python (Thư mục: `04_Python`)
Khi dữ liệu quá lớn hoặc cần phân tích phức tạp, tự động hóa, Python sẽ tỏa sáng.
- [ ] Cú pháp cơ bản, các kiểu dữ liệu, vòng lặp, hàm.
- [ ] Thư viện **Pandas** (Thao tác, làm sạch, và biến đổi dữ liệu - Data Manipulation).
- [ ] Thư viện **NumPy** (Tính toán toán học).
- [ ] Thư viện **Matplotlib** & **Seaborn** (Trực quan hóa dữ liệu bằng code).

## 🧮 5. Xác suất thống kê (Thư mục: `05_Statistics`)
Tư duy thống kê giúp bạn đưa ra nhận định chính xác và không bị dữ liệu "đánh lừa".
- [ ] Thống kê mô tả (Descriptive Statistics: Mean, Median, Mode, Variance, Standard Deviation).
- [ ] Phân phối dữ liệu (Normal Distribution, Skewness).
- [ ] Thống kê suy diễn (Inferential Statistics).
- [ ] A/B Testing và Kiểm định giả thuyết (Hypothesis Testing, p-value).

## 💼 6. Thực hành Projects (Thư mục: `06_Projects`)
Học phải đi đôi với hành. Xây dựng Portfolio cá nhân để ghi điểm với nhà tuyển dụng.
- [ ] **Project 1:** Phân tích dữ liệu bằng Excel (Sales Analysis).
- [ ] **Project 2:** Phân tích dữ liệu bằng SQL (Customer Churn/Retention).
- [ ] **Project 3:** Dashboard bằng Power BI/Tableau.
- [ ] **Project 4:** Phân tích chuyên sâu bằng Python (EDA - Exploratory Data Analysis).

- Mỗi khi tìm được sách, file PDF, hoặc bài tập thực hành mới, hãy lưu trực tiếp vào các thư mục tương ứng.
