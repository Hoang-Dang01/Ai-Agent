# Thư mục lưu trữ N8N Workflows

Đây là nơi để lưu trữ các luồng (Workflows) n8n đã xuất ra (export) hoặc dùng để dự phòng (backup) cho Hệ thống Trợ Lý Học Tập.

## Hướng dẫn sử dụng:
1. **Xuất luồng (Export):** Khi anh cấu hình xong một luồng trên phần mềm diện UI của n8n, hãy chọn `Menu > Download` để tải file `.json` xuống.
2. **Lưu trữ:** Kéo thả tệp `.json` đó vào trong thư mục này. Ví dụ: `tro-ly-hoc-tap-main-flow.json`.
3. **Phục hồi (Import):** Khi cần triển khai sang môi trường mới hoặc vô tình làm hỏng trên n8n, anh có thể vào giao diện n8n, chọn `Menu > Import from file` và trỏ vào file lưu trong thư mục này.

> **Mẹo:** Việc lưu các tệp .json này kèm với mã nguồn Github/Git giúp quản lý phiên bản của Workflow (luồng) rất tốt, anh có thể theo dõi xem hôm nay mình đã sửa/kéo thêm node nào so với tuần trước.
